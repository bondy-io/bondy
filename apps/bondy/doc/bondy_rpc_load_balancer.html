    bondy_rpc_load_balancer (bondy v1.0.0-beta.16) View Source

    This module implements a distributed load balancer, providing the different load balancing strategies used by bondy_dealer to choose the Callee and Procedure to invoke when handling a WAMP Call.

    At the moment the load balancing state is local and not replicated across the nodes in the cluster. However, each node has access to a local replica of the global registry and thus can load balance between local and remote Callees.

    ## Supported Load Balancing Strategies

    Bondy supports all WAMP Basic and Advanced Profile load balancing strategies for Shared Registrations and extends those with additional strategies.
